# Project Reports & Slides

This directory or folder contains the formal documentation and presentation(final) deliverables for the **ResonAI** project (Deep Learning course).

### 1. Final Report Research Paper (Deep Learning)
* **File:** `Final Report Paper DL(1).pdf`
* **Content:** A comprehensive and complete report or research paper, it covers the 1D-CNN architecture design, data preprocessing, model evaluation (74.39% accuracy), and deployment details. 

### 2. Presentation Slides (PPT)
* **File:** `Presentation_Slides_DL.pdf`
* **Content:** A visual summary of the project, covering the introduction, problem background, dataset, methodology, technical architecture, evaluations, demo video and reflections/conclusions.

## Report Highlights
The final report discusses:
* **Methodology:** Implementation of a lightweight 1D-CNN for real-time pitch estimation.
* **Experiments:** Comparison between the baseline model and data augmentation techniques on the MIR-1K dataset.
* **Deployment:** Details on the serverless deployment using Gradio and Hugging Face Spaces.
